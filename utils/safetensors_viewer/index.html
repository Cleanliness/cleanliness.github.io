Safetensors Viewer

Prints out the header of a single safetensors file. Useful for getting a basic idea of model architecture. Use the url from 'copy download link' for any safetensors file and paste it here. Any download link should work provided it's not blocked by CORS.

Some examples:

  • https://huggingface.co/stabilityai/stable-diffusion-2-1/resolve/main/v2-1_768-ema-pruned.safetensors
